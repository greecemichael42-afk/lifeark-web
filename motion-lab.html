معمل الموشن

كل أنظمة الموشن الجديدة في صفحة واحدة للمراجعة قبل النقل للصفحات الحقيقية. مفيش حاجة هنا بتلمس الموقع اللايف.

System 01

فيديو بيتحرك مع السكرول

انزل بالسكرول جوه المشهد — الفيديو بيتقدم ويرجع معاك فريم بفريم. ده نفس النظام اللي هيشغّل رحلة «من العاصفة للفجر» على الهوم.

🟢 جاهز — فيديو «العاصفة → الفجر» الحقيقي (Veo مجانًا، العلامة المائية اتشالت، all-keyframe): عاصفة وفانوس → السفينة تتجلى → فجر ذهبي وحمامة وقوس قزح

في قلب العاصفة

لما كل حاجة بتبقى غامقة ومش مفهومة

الفانوس لسه شغّال

مهارة واحدة صغيرة ممكن تمسّك الدفة

والفجر بييجي

الرحلة ليها بر تاني — ودي مش وعود، دي منهجية

↓ استمر في السكرول — الفيديو بيتحرك معاك
System 02

مشهد مجسَّم بالطبقات

نفس صورة السفينة المولّدة جديد — بس بعمق حقيقي: الطبقات بتتحرك بسرعات مختلفة مع السكرول، والمشهد بيميل مع حركة الماوس.

🟢 جاهز — الصورة الأم مولّدة فعلاً بهيجس فيلد (Cinema Studio 2.5)

System 03

الكروت الحيّة — الركائز الأربعة

حط الماوس على أي كارت: توهّج الفانوس بينبض والصورة بتقرّب بهدوء. لما نولّد السينماجرافات، نفس الكارت هيشغّل فيديو loop بدل الصورة — النظام جاهز.

🟢 كارت الجلسات بقى حي فعلًا (Veo مجاني — قرّب الماوس منه!) · الباقي ٣ كروت بنفس الطريق المجاني

System 04

باند القصيدة الحي

نفس باند «أعلم طريقتي في النجاة» — الصورة بتتنفس: زوم بطيء جدًا + شعاع ضوء بيعدي، والقصيدة نص HTML فوقها زي ما هي.

🟢 معتمد نهائي (قرار مايكل 2026-07-09) — صورة مايكل تفضل + الحركة السينمائية CSS؛ مفيش فيديو محتاج هنا

System 05

السفينة المجسَّمة 3D

حرّك الماوس فوق الإطار — ده «تذوّق» بميل CSS ثلاثي الأبعاد. النسخة الحقيقية: موديل GLB كامل من صورة السفينة تقدر تلفّه بإيدك 360°.

⏳ مستني كريديت — توليد الموديل الـ 3D (image_to_3d) + عارض model-viewer محلي

ليه دي حاجة محدش عاملها؟

مفيش عيادة نفسية في المنطقة عندها رمز علامتها كمجسم ثلاثي الأبعاد حي جوه الموقع. الزائر يلف السفينة بإيده — إحساس «المجسم» الحقيقي اللي بندوّر عليه من الأول.

التنفيذ: توليد GLB من صورة الهيرو ← عارض ثلاثي الأبعاد محلي (بدون CDN) ← سقوط مريح لصورة ثابتة على الأجهزة الضعيفة.
